📋 How to Use This Tool

Step 1: Export your CSVs from Google Workspace Admin Console, then upload

  • Chromebook Devices CSV: Admin Console → Devices → Chrome devices → Export. Upload the resulting file (filename is a UUID, e.g. 824d4ee6-…csv).
  • Users CSV: Admin Console → Directory → Users → Download users → "All user info". Upload the resulting User_Download_<date>_<time>.csv file as-is. The tool keeps only users whose Status is Active; any further OU-based filtering is controlled by Settings above.

Step 2: Filter your data (optional)

  • Search School: Type part of a school name to focus on specific schools
  • Filter by Model: Check boxes to see only specific Chromebook models (e.g., "Acer Chromebook")
  • Filter by Year: Check boxes to see only devices made in certain years
  • Filter by OS Version: Check boxes to see devices running specific Chrome OS versions

Step 3: View your results

  • Charts show: Device counts, student counts, or student-to-device ratios by school
  • Tables show: Detailed breakdown by school and individual device information
  • Color coding: Green = Good, Yellow = Acceptable, Red = High (thresholds set in Settings)
