Settings— adapt this tool to your organization

Settings are stored in this browser (localStorage) and never leave your machine. Click Save to apply; if CSVs are already loaded they will be re-processed with the new values.

Tip. Place your config at data/config.json (git-ignored). It auto-loads when localStorage is empty. If the browser blocks the auto-fetch — common with file:// — use Load from file instead. See example.config.json for the format.
